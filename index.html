Projects

01

3D Printer Design and Build

Designed and built a coreXY 3D printer running on Klipper firmware that rivals the performance of top-level consumer models.

  • Embedded Systems
  • Python
  • Controls
View Details →

This printer serves as a customizable platform for testing 3D printer performance improvements. Implemented an ADXL345 accelerometer-based input shaping algorithm which avoids resonance peaks to reduce print artifacts and improve surface finish at high speeds. Analyzed power consumption and thermal performance of the stepper drivers under various print loads, optimizing cooling to maintain consistent performance during long prints.

02

EV Battery Pack Thermal Model

Built a Simulink thermal simulation for a lithium-ion battery pack under drive-cycle loading. Compared passive vs. active cooling strategies and plotted cell temperature distributions against SAE J2288 profiles.

  • MATLAB
  • Simulink
  • Thermal Modeling
View Details →

Created a multi-domain thermal model that tracked cell temperatures across charge/discharge cycles. I compared passive and active cooling to expose hot spots and quantify how design changes improved pack longevity.

03

Conveyor Line PLC Automation

Programmed ladder logic for a three-station sorting conveyor using Allen-Bradley MicroLogix 1100. Integrated photoelectric sensors, VFD speed control, and a basic HMI touchscreen for operator monitoring.

  • PLC
  • Ladder Logic
  • VFD
  • HMI
View Details →

Developed logic to sequence conveyor stations, manage material flow, and support fault recovery. The implementation included safety interlocks, sensor-driven step sequencing, and an operator HMI that simplified machine status checks.

About Me

Engineer,
problem-solver,
tinkerer.


I'm a third-year Electrical Engineering student at Clarkson University in Potsdam, NY. My coursework focuses on control systems and power electronics, and I spend a lot of time outside class poking at motor drives and writing embedded firmware.

My longer-term interest is in making electric vehicles smarter — better thermal management, tighter torque control, and the kind of real-time feedback loops that separate good EVs from great ones.

When I'm not in the lab I'm probably hiking the Adirondacks or taking something apart to see how it works.
